# Piedra, Papel o Tijera 🎮

Aplicación web del clásico juego piedra, papel o tijera.

## 🚀 Demo

https://tu-usuario.github.io/tu-repo/

## 🧠 Conceptos utilizados

* Router
* State
* Pages
* Components

## 🎯 Funcionalidad

El usuario puede elegir entre piedra, papel o tijera.
La computadora genera una jugada aleatoria y se muestra el resultado:

* Ganar
* Perder
* Empatar

## 🛠️ Tecnologías

* HTML
* CSS
* JavaScript

## 📦 Instalación

1. Clonar el repositorio
2. Abrir el proyecto
3. Ejecutar con Live Server o similar
